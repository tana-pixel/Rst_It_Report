# 📊 IT Department Performance & System Dashboard

ยินดีต้อนรับสู่หน้ารวบรวมข้อมูลและรายงานผลการดำเนินงานของฝ่ายสารสนเทศ (IT Dashboard) สำหรับใช้ในการติดตาม ประเมินผล และสนับสนุนการตัดสินใจในการพัฒนาโครงสร้างพื้นฐานด้านเทคโนโลยีขององค์กร

## 🎯 วัตถุประสงค์
1. **Performance Tracking:** ติดตามสถิติและประสิทธิภาพการทำงานของระบบสารสนเทศ
2. **System Optimization:** รวบรวมข้อมูลเพื่อใช้วิเคราะห์และปรับปรุงระบบให้ดียิ่งขึ้น
3. **Data-Driven Decisions:** ใช้ข้อมูลสนับสนุนการตัดสินใจของผู้บริหารในการวางแผนพัฒนา IT
